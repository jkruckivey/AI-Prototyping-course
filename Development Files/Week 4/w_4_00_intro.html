 What is Generative AI?

Definition

Generative AI is a subset of AI focused on creating new content—text, images, audio, video, or code—based on patterns learned from training data. Unlike traditional ML that classifies or predicts, generative AI creates.

Examples include:

  • Text generation (ChatGPT, Claude, Gemini)
  • Image creation (DALL-E, Midjourney, Stable Diffusion)
  • Code generation (GitHub Copilot, Amazon CodeWhisperer)
  • Music and audio synthesis

How is it different from traditional ML?

Traditional ML vs. Generative AI
Aspect Traditional ML Generative AI
Primary Task Classification, prediction, clustering Content creation, generation
Output Labels, categories, numeric predictions New text, images, code, audio
Training Data Task-specific labeled datasets Massive diverse datasets (often unlabeled)
Model Size Thousands to millions of parameters Billions to trillions of parameters
Adaptability Single task, needs retraining for new tasks Multi-task, can adapt via prompting

Common NLP tasks

Natural Language Processing (NLP) tasks that Generative AI excels at:

  • Text summarization: Condensing long documents into key points
  • Translation: Converting text between languages
  • Question answering: Responding to queries based on context
  • Sentiment analysis: Determining emotional tone of text
  • Named entity recognition: Identifying people, places, organizations
  • Text generation: Creating original content from prompts
  • Code generation: Writing programming code from descriptions
