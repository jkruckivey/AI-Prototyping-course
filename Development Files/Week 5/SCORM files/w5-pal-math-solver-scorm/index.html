One-Shot Prompting: Teaching the LLM

Step 1: Provide One Example

Q: Roger has 5 tennis balls. He buys 2 cans of tennis balls. Each can has 3 balls. How many does he have now?

# Roger started with 5 tennis balls tennis_balls = 5 # Two cans of tennis balls are bought, each can has 3 balls bought_balls = 2 * 3 # The answer is tennis_balls + bought_balls answer = tennis_balls + bought_balls # answer = 5 + 6 = 11
Step 2: Ask Your Question

Q: The bakers at Beverly Hills Bakery baked 200 loaves of bread on Monday morning. They sold 93 loaves in the morning and 39 loaves in the afternoon. A grocery store returned 6 unsold loaves. How many loaves did they have left?

Step 3: LLM Follows the Pattern
# The bakers baked 200 loaves loaves_baked = 200 # They sold 93 in the morning sold_morning = 93 # They sold 39 in the afternoon sold_afternoon = 39 # A grocery store returned 6 loaves returned = 6 # The answer is loaves_baked - sold_morning - sold_afternoon + returned answer = loaves_baked - sold_morning - sold_afternoon + returned # answer = 200 - 93 - 39 + 6 = 74
‚úÖ Correct Answer: 74 loaves
The LLM followed the one-shot pattern and generated correct Python code!

Why One-Shot Prompting Works

üß† The Key Insight

LLMs are pattern matchers, not calculators. By showing them ONE example of the desired reasoning pattern in Python-style comments and variables, they can replicate that pattern for similar problems.

What the One-Shot Example Teaches:

  1. Structure: Break problem into steps with comments
  2. Variables: Create variables for each piece of information
  3. Operations: Show how to combine variables mathematically
  4. Format: Use Python syntax that the code interpreter can execute

Modern LLMs: No Prompt Engineering Needed

Today's LLMs (ChatGPT, Claude, Gemini) are pre-trained to use code interpreters. You can simply ask the question‚Äîthe LLM will:

  • Detect that computation is needed
  • Automatically generate Python code
  • Execute it via built-in interpreter
  • Return the correct answer

Pro tip: If the LLM isn't using code when it should, explicitly request: "Please use Python code interpreter to solve this."
