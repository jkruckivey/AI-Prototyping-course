 Key Takeaways

LLMs Have Evolved Beyond Text

  1. From predictors to agents: LLMs now execute code, access tools, and perform actions
  2. Orchestration libraries: Enable seamless integration between natural language and Python execution
  3. Natural language as programming: English (or any language) becomes a valid programming interface
  4. Democratized data science: Anyone with domain knowledge can now build ML models

The Customization Stack

  1. Prompt engineering: Zero/one/few-shot learning, prompt patterns (Week 4)
  2. RAG (Knowledge bases): Upload documents, automatic retrieval and augmentation
  3. Custom GPTs: Personas, guardrails, capabilities, actions—no code required
  4. Fine-tuning: Retrain weights for consistent style, format, or specialized behavior
  5. Combine approaches: Production systems often use all four layers

Trust, but Verify

  • LLMs are powerful assistants, not infallible experts
  • Hallucinations happen: Always validate critical outputs
  • Domain knowledge essential: Your expertise guides and validates the AI
  • Think "AI intern": Knows how to code, needs clear instructions and supervision
  • Break tasks into steps: Incremental progress reduces errors
  • Review generated code: Understand what the AI is doing before trusting results

 Practical Next Steps

Practice data science with code interpreter

  • Upload your own datasets to ChatGPT's code interpreter
  • Recreate past data science projects using natural language
  • Build ML models (regression, classification) without writing code
  • Generate visualizations and export to PowerPoint/Excel
  • Save models as PKL files for reuse

Build a custom GPT

  • Identify a repetitive task in your work that could be automated
  • Create a custom GPT with clear instructions and persona
  • Upload relevant knowledge base documents (PDFs, policies, guides)
  • Test with colleagues, gather feedback, iterate
  • Share with your team or publish to GPT store

Experiment with fine-tuning

  • Identify a task where consistent format or style is critical
  • Collect 100+ examples (or use ChatGPT to generate training data)
  • Format as JSONL (task or conversational)
  • Create fine-tuning job in OpenAI Dashboard
  • Test your custom model in Playground
  • Compare performance to prompt-engineered baseline
