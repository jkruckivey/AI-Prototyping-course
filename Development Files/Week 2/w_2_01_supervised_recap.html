 Supervised Learning Recap

What this page covers

Foundations you’ll use all week: features → target mapping, difference between classification and regression, and key terms like parameters, hyperparameters, and dimensionality.

 What is Supervised Learning?

Features → Target mapping

We learn a function that maps input features (columns) to a target (output). Training adjusts model parameters to minimize a chosen loss, aiming to generalize to unseen data.

Classification

Predicts a category (e.g., approve/deny, churn/no-churn, dog/cat). Outputs are discrete labels or probabilities, with a decision threshold applied for the final class.

Regression

Predicts a number (e.g., price, demand, time-to-complete). We evaluate with numeric error metrics and often visualize residuals to sanity-check fit.

 Key Terms You’ll See All Week

Parameters vs Hyperparameters

  • Parameters: learned values inside the model (e.g., coefficients, weights, intercepts).
  • Hyperparameters: configuration chosen before training that controls learning (e.g., learning rate, regularization strength, K in KNN).

Dimensionality

The number of input features. More dimensions can improve accuracy or introduce noise/leakage—use validation to confirm real gains.

Train/Validation Splits

Always separate development and evaluation. For time‑series or sequential data, use time‑aware splits to avoid peeking into the future.

Helpful Tip

Pick metrics that fit the target type: classification ⇒ precision/recall/ROC; regression ⇒ MAE/MSE/RMSE. Tie choices to business cost.
