🔗 Widget Combiner Tool

Combine Canvas HTML pages with their embedded widgets into a single, self-contained file. This tool fetches widget content from iframes and inlines them directly into the HTML, perfect for SCORM packaging.

📋 How to Use

  1. Upload Canvas HTML: Select a Canvas HTML file that contains iframe widgets
  2. Process: The tool will detect all embedded widgets and fetch their content
  3. Combine: Widgets will be inlined into the HTML, replacing iframes
  4. Download: Get the combined HTML file ready for SCORM conversion
