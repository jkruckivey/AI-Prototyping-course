The Classification Problem

A grocery store wants to automatically classify fruits as apples or oranges using two features:

  • Color Score: 0 (green) to 10 (red)
  • Weight: In grams
üçé Apples: Higher color scores (red), varying weights
üçä Oranges: Lower color scores (orange), typically heavier
2

Why Not Linear Regression?

Linear regression predicts continuous values, but we need discrete classes (Apple or Orange).

Linear: Could predict "1.5 apples" or "-0.3 oranges" ‚ùå
Logistic: Predicts probabilities between 0 and 1 ‚úÖ

We need probabilities: "80% chance this is an apple"

3

The Sigmoid Function

Instead of a straight line, logistic regression uses an S-shaped sigmoid curve:

üìà S-shaped curve - smooth transition between classes
üìä Bounded 0-1 - outputs valid probabilities
üéØ Threshold at 0.5 - decision boundary

üìö Key Concepts

Classification vs Regression

Classification: Predicts discrete categories (Apple/Orange)
Regression: Predicts continuous values (price, height)
Logistic regression is for classification despite its name!

Sigmoid Function

The S-shaped curve maps any input to a value between 0 and 1, making it perfect for probabilities. Formula: 1 / (1 + e^(-x))

Decision Boundary

The threshold (usually 0.5) where we decide between classes. Above threshold = Class 1, below = Class 0.

Training Process

Uses Maximum Likelihood Estimation or gradient descent to find the best sigmoid curve parameters that minimize classification errors.
